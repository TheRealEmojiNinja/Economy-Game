What Works:

The province generation works perfectly. 
You are also able to construct factories and mines using money for now, and it finishes production in the expected time. 
You are able to progress through the day properly and it updates the economic output (money).

What Is Unfinished:

When progressing through the days, it does not update the iron and coal output.
Coal needs to be used daily by factories to continue production, otherwise they stop producing.
Iron needs to be used in the construction of new factories.
